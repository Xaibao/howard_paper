====================================================
  Spectral Monitor 系統啟動說明
====================================================

需要開 3 個終端機，依序啟動：


【Terminal 1】Flask API（模型推論 + LLM）
----------------------------------------------------
conda activate spectral
cd /home/t1204-3060/Howard/spectral_monitor
python src/app.py

等到看到：
  * Running on http://0.0.0.0:5000
才算啟動完成。


【Terminal 2】RTMP 串流（無人機直播）
----------------------------------------------------
cd /home/t1204-3060/Howard/spectral_monitor
bash rtmp-server/start_all.sh

等到看到：
  === 全部啟動完成 ===
才算啟動完成。


【Terminal 3】Angular 儀表板（網頁介面）
----------------------------------------------------
cd /home/t1204-3060/Howard/spectral_monitor/frontend
npx ng serve --host 0.0.0.0 --port 4200

等到看到：
  Application bundle generation complete.
才算啟動完成。


====================================================
  開啟瀏覽器
====================================================

  http://100.108.78.9:4200


====================================================
  關閉系統
====================================================

每個 Terminal 按 Ctrl+C 即可關閉。


====================================================
  注意事項
====================================================

- Terminal 1 一定要先啟動，其他兩個順序不限
- 樹莓派 Samba 需要另外手動掛載：
    sudo mount -t cifs //100.74.109.10/share /home/t1204-3060/rpi_share -o username=pi,password=t1204
- 無人機推流網址：rtmp://100.108.78.9:1935/live/stream
